FileTracker
===========

A tracker of changes on files

## 概要
FileTrackerは、ファイルに加えられた変更を追跡し、必要に応じて復元するソフトウェアです。

## 詳細
[Wiki](https://github.com/paralleltree/FileTracker/wiki)を参照してください。
